## ⭐ Flagship: MIRA — Multi-AI Orchestrator

**The intelligent brain that delegates to specialized AI workers.**

**Problem:** Managing 5+ AI tools manually = confusion, wasted tokens, inconsistent output.

**Solution:** MIRA (Claude) as architect. ChatGPT handles coding. Gemini analyzes data. Grok researches trends. All orchestrated transparently, cost-optimized.

### Three Proven Workflows

**1. Content Generation Pipeline**
```
Outline → Draft → Review → Publish
MIRA   → ChatGPT → Editor → Ready
```
Perfect for: Blog articles, curriculum guides, whitepapers, educational content.

**2. Code Review & Optimization**
```
Analyze → Optimize → Document → QA
MIRA   → ChatGPT  → ChatGPT  → Editor
```
Perfect for: Legacy code refactoring, performance optimization, documentation.

**3. Research & Intelligence Synthesis**
```
Plan → Gather → Analyze → Synthesize → Publish
MIRA → Grok  → Gemini  → ChatGPT   → Editor
```
Perfect for: Market research, trend analysis, competitive intelligence, data synthesis.

## 🔧 Getting Started with CrewAI

### Installation

```bash
pip install crewai anthropic openai google-genai x-api
```

### Environment Setup

```bash
export ANTHROPIC_API_KEY="your_claude_key"
export OPENAI_API_KEY="your_chatgpt_key"
export GOOGLE_API_KEY="your_gemini_key"
export X_API_KEY="your_grok_key"
```

### Run Example Workflows

```bash
cd crewai
python orchestrator.py

# Choose workflow:
# 1. Content Generation (outline → draft → review)
# 2. Code Review (analyze → optimize → document)
# 3. Research Synthesis (plan → gather → analyze → synthesize)
```

## 💡 How MIRA Works

**Cost-first orchestration without sacrificing quality:**

1. **Architect** (MIRA/Claude) breaks down the task
2. **Delegate** based on capability + cost:
   - ChatGPT for coding (cheaper than Claude)
   - Gemini for images (free tier)
   - Grok for real-time data (X API)
3. **Synthesize** — Editor QA ensures output quality
4. **Repeat** — Learn from results, optimize next time

**Result:** Faster delivery, lower token spend, and transparent, auditable workflows.
